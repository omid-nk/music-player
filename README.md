A simple **music player** built using **Vite**, **TailwindCSS**, and **vanilla JavaScript**.  
Supports basic controls like **play/pause**, **next/previous**, **mute/unmute**, and **progress seeking**.

## Features

- Responsive design using TailwindCSS
- Play, pause, skip, and mute/unmute controls
- Audio progress bar with time display
- Display artist name, song name, and cover art
- Works both locally (Vite dev) and on GitHub Pages
